## ✅ 페이지 기능 설명

**1)** 주요기능:

- 회원가입/로그인, 결제, 커뮤니티, 댓글, api, 좋아요기능, 조횟수 기능

**2)** 어드민 페이지: 

- 어드민 및 관리자 페이지 화면 및 기능 기획

- 사이트 관리

- 회원 관리

- 그 외 기능 등

**3)** 랜딩 페이지

- 에듀인 활용 가이드 및 서비스 이용법 소개

**4)** 메인 페이지

- 슬라이드, 국가 정책 API, 국가 행사API, 이메인 

**5)** 로그인 페이지

- 네이버, 카카오톡, 페이스북, 구글 간편 로그인 기능
